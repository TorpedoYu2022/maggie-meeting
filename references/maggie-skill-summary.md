# Maggie 📅 技能总结

> 最后更新：2026-03-29

---

## 一、约会议（核心技能）

**能力：** 创建腾讯会议、冲突检测（3账号选号）、时间戳校验

**流程：**
1. 收到约会议请求（群里艾特或私聊）
2. 确认信息：主题、客户、时间（默认1小时不追问）
3. 查冲突（本地表 + API 查3个账号）
4. 选空闲账号创建
5. 回显：📅 日期时间 + 主题 + 会议号 + 入会链接

**3个账号：**
- 账号1/2：个人账号，无时长限制
- 账号3：企业账号，40分钟限制

**关键规则：**
- 时间戳必须用脚本算，禁止心算
- time_zone 必传 Asia/Shanghai
- 创建前必须校验时间戳反转为可读时间

---

## 二、会议纪要（自动触发）

**能力：** 获取转写/智能纪要、整理格式、发邮件

**客户会议（SPCA 模型）：**
- S 背景 → P 痛点 → C 共识 → A 行动

**内部会议（标准格式）：**
- 主题、参会人、讨论内容、决议、行动项

**技术细节：**
- 会议号（meeting_code）≠ 内部ID（meeting_id）
- 查录制/纪要必须用 meeting_id
- start_time 往前推24小时，end_time 往后推1小时

---

## 三、Salesforce 表单（自动提交）

**能力：** 生成预填表单链接、预约人确认后自动写入 SF

**流程：**
1. Maggie 生成一次性 token（24h有效）
2. 拼接预填表单链接（含纪要）
3. 发邮件给预约人（抄送 T Sir）
4. 预约人检查补充 → 点击提交
5. 表单服务自动写入 SF CSM__c 对象

**表单服务：** `maggie_csm_server.py`，端口 9090
**SF 对象：** CSM__c，17个字段映射
**自动处理：** 查找 Account ID、继承 CSM 负责人

### 字段映射表

| 表单字段 | Salesforce 字段 | 类型 |
|----------|----------------|------|
| 客户 | ExistedAccount__c | 查找(客户) |
| 服务时间 | date__c | 日期 |
| 拜访方式 | VisitMethod__c | 选项列表 |
| 服务大类 | ActivityType__c | 选项列表 |
| 参会人（客户方） | attendAcount__c | 文本区 |
| 参会人（我方） | AttendMe__c | 选项列表(多项) |
| KP参与 | KP__c | 选项列表 |
| 是否深度服务 | DeepService__c | 选项列表 |
| 准备工时 | PrepareTime__c | 数字 |
| 服务工时 | Effort__c | 数字 |
| 续约判断 | Renewal__c | 选项列表 |
| 服务详情 | ServiceDetail__c | 文本(255) |
| SPCA 纪要 | meetingminutes__c | 长文本区 |
| SPCA 共识 | PromiseTodo__c | 长文本区 |
| SPCA 行动 | LaterPlan__c | 长文本区 |
| 客户反馈 | Feedback__c | 长文本区 |
| 未解决风险 | UnresolveIssueRisk__c | 长文本区 |

## 六、完整工作流程

### 阶段一：预约会议

| # | 执行者 | 动作 |
|---|--------|------|
| 1 | CSM/同学 | 在群里艾特 Maggie 约会议 |

| 6 | Maggie | 回显完整信息 |

| 13 | Maggie | 拼接预填表单链接 |
| 14 | Maggie | 发邮件：纪要+链接，抄送T Sir |

|---|--------|------|

### 阶段四：自动写入 Salesforce

| # | 执行者 | 动作 |
|---|--------|------|

| 23 | 表单服务 | 写入 CSM__c |
